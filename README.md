🐉 Dragon Diet Plan

📖 Description

The Dragon Diet Plan is a simple HTML webpage that shows a creative daily meal plan for dragons.
It includes meals like breakfast, lunch, dinner, and snacks, and explains the nutrition dragons need to stay strong and breathe fire.

✨ Features

- Colorful webpage design
- Different sections for meals
- Ordered and unordered lists
- Nutrition description list
- Inline CSS styling (background color, borders, spacing)
- External link for healthy eating tips

🛠 Technologies Used

- HTML5
- Inline CSS

📂 Project Purpose

This project was created to practice basic HTML tags and inline CSS styling while building a fun webpage.
